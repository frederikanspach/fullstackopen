# Full Stack Open 2026

Dieses Repository enthält meine Lösungen und Projekte für den [Full Stack Open Kurs](https://fullstackopen.com/de/) der Universität Helsinki. 

Nach einer längeren Pause (2008–2025) nutze ich diesen Kurs für einen "Deep Dive" in das moderne Web-Development.

## 🛠 Tech Stack
- **OS:** Linux Mint
- **Editor:** VS Code
- **Frontend:** React, Redux, Vite
- **Backend:** Node.js, Express
- **Datenbank:** MongoDB, PostgreSQL
- **DevOps:** Docker, GitHub Actions, Debian Testing (Server)
